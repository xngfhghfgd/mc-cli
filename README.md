# mc-cli

Minecraft 1.18.2 命令行客户端。

## 功能

- 连接 Minecraft 1.18.2 服务器
- 支持离线模式登录
- 聊天发送与接收
- 基础移动控制
- `ping` 查询服务器状态

## 用法

```bash
./mc <服务器地址> [端口] [用户名]
./mc ping <服务器地址> [端口]
```

## 基础命令

- `/w` `/a` `/s` `/d`：持续移动
- `/stop`：停止移动
- `/jump` 或 `/j`：跳跃一次
- `/q`：退出

## 说明

当前版本仅支持离线模式服务器；如果服务器要求正版验证，会提示登录被拒绝。
