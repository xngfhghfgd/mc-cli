# mc-cli

`mc-cli` 是一个面向 Minecraft 1.18.2 的命令行客户端。它可以从终端连接服务器、发送聊天消息、接收聊天内容，并提供一些基础移动控制，适合做自动化测试、轻量控制或协议实验。

> 说明：当前版本仅支持离线模式服务器；如果服务器要求正版验证，客户端会提示登录被拒绝。

## 特性

- 连接 Minecraft 1.18.2 服务器
- 支持离线模式登录
- 支持聊天收发
- 支持基础移动控制
- 支持 `ping` 查询服务器状态
- 纯命令行交互，适合脚本化使用

## 快速开始

```bash
./mc <服务器地址> [端口] [用户名]
```

例如：

```bash
./mc localhost 25565 HermesBot
```

如果不指定端口，默认使用 `25565`。如果不指定用户名，默认使用 `HermesBot`，也可以通过环境变量 `MC_USERNAME` 覆盖。

## 查询服务器状态

```bash
./mc ping <服务器地址> [端口]
```

示例：

```bash
./mc ping localhost 25565
```

输出会包含：

- 在线状态
- 延迟
- MOTD
- 当前玩家数
- 服务器版本信息

## 基础控制命令

在客户端交互中可以使用以下命令：

- `/w` 或 `/forward`：持续向前移动
- `/a` 或 `/left`：持续向左移动
- `/s` 或 `/back`：持续向后移动
- `/d` 或 `/right`：持续向右移动
- `/stop` 或 `/x`：停止移动
- `/jump` 或 `/j`：跳跃一次
- `/q`：退出客户端

普通文本会直接发送为聊天消息。

## 文件结构

- `cli.py`：命令行入口
- `client.py`：客户端连接、登录、聊天和移动逻辑
- `protocol.py`：Minecraft 协议编码/解码实现
- `mc`：可执行入口脚本
- `debug_send.py`：调试辅助脚本

## 安装与运行

### 直接运行

```bash
chmod +x mc
./mc localhost 25565
```

### 作为脚本执行

```bash
python3 cli.py localhost 25565
```
